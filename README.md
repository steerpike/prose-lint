# Prose Lint

A prose linting plugin for Obsidian.

- Coming soon...

## Installation

1. Copy the plugin files to your vault's `.obsidian/plugins/prose-lint/` directory
2. Reload Obsidian or enable the plugin in settings
3. Start using the plugin!

## Development

To build this plugin:

```bash
npm install
npm run build
```

To develop with hot reload:

```bash
npm run dev
```
